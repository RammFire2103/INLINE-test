## Требования

Для запуска проекта на своем компьютере необходимо наличие следующих программ:

- [Node.js](https://nodejs.org/en/) версии 16 и выше.
- [npm](https://www.npmjs.com/) для установки зависимостей.

## Установка и запуск проекта
